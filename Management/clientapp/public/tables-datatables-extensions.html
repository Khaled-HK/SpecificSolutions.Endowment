التمرير المتقدم (Scrollable Tables)

تحكم كامل في التمرير الأفقي والعمودي للجداول الكبيرة

الميزات:
  • تمرير عمودي بارتفاع محدد
  • تمرير أفقي للجداول العريضة
  • تحسين الأداء للبيانات الكبيرة
  • تحكم في سرعة التمرير
كود التطبيق:
$('#example').DataTable({
    scrollY: '400px',
    scrollX: true,
    scrollCollapse: true,
    paging: false
});

العنوان الثابت (Fixed Header)

عنوان الجدول يبقى ثابتاً أثناء التمرير العمودي

الميزات:
  • عنوان ثابت أثناء التمرير
  • تحسين تجربة المستخدم
  • سهولة القراءة
  • تحكم في موضع العنوان
كود التطبيق:
$('#example').DataTable({
    fixedHeader: true,
    scrollY: '400px',
    scrollCollapse: true
});

الأعمدة الثابتة (Fixed Columns)

تثبيت أعمدة معينة أثناء التمرير الأفقي

الميزات:
  • تثبيت أعمدة من اليسار أو اليمين
  • سهولة المقارنة
  • تحسين الملاحة
  • مرونة في التحكم
كود التطبيق:
$('#example').DataTable({
    fixedColumns: {
        leftColumns: 2,
        rightColumns: 1
    },
    scrollX: true
});

تحديد الصفوف (Row Selection)

تحديد متعدد للصفوف مع عمليات جماعية

الميزات:
  • تحديد فردي أو متعدد
  • تحديد الكل مع خانة واحدة
  • عمليات جماعية
  • تتبع الصفوف المحددة
كود التطبيق:
$('#example').DataTable({
    select: {
        style: 'multi',
        selector: 'td:first-child'
    },
    columnDefs: [{
        targets: 0,
        checkboxes: {
            selectRow: true
        }
    }]
});

  :headers="headers"
  :items="items"
  fixed-header
  height="400"
  show-select
  class="text-no-wrap"
>
  <template #item.actions="{ item }">
    <!-- أزرار الإجراءات -->
  </template>
</VDataTable>

نصائح مهمة:
  • استخدم sticky: true للأعمدة الثابتة
  • حدد عرض ثابت للأعمدة المثبتة
  • استخدم fixed-header للعنوان الثابت
  • أضف height لتفعيل التمرير العمودي
